# @ncthub/ai

AI utilities and helpers for Tuturuuu Platform.

## Installation

```bash
npm install @ncthub/ai
# or
yarn add @ncthub/ai
# or
bun add @ncthub/ai
```

## Development

```bash
# Install dependencies
bun install

# Run tests
bun run test
```
