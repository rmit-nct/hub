# @ncthub/ai

AI utilities and helpers for Tuturuuu Platform.

## Installation

```bash
npm install @ncthub/ai
# or
yarn add @ncthub/ai
# or
pnpm add @ncthub/ai
```

## Development

```bash
# Install dependencies
pnpm install

# Run tests
pnpm test
```
