# @ncthub/supabase

Supabase client and utilities for Tuturuuu Platform.

## Installation

```bash
npm install @ncthub/supabase
# or
yarn add @ncthub/supabase
# or
pnpm add @ncthub/supabase
```

## Development

```bash
# Install dependencies
pnpm install

# Run tests
pnpm test
```
