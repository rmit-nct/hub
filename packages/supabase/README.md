# @ncthub/supabase

Supabase client and utilities for Tuturuuu Platform.

## Installation

```bash
npm install @ncthub/supabase
# or
yarn add @ncthub/supabase
# or
bun add @ncthub/supabase
```

## Development

```bash
# Install dependencies
bun install

# Run tests
bun run test
```
